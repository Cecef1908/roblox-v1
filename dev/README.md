# dev/ — Dossier Go Dev

Ce dossier contient tout le necessaire pour le developpement actif de Gold Rush Legacy.

## Fichiers

| Fichier | Contenu |
|---------|---------|
| **ROADMAP.md** | Plan de developpement en 20 sprints, 4 phases |
| **AUDIT.md** | Audit complet du codebase — ce qui marche, ce qui manque |
| **SPRINT-TRACKER.md** | Suivi sprint par sprint (cocher au fur et a mesure) |

## Comment utiliser

1. Lire **AUDIT.md** pour comprendre l'etat actuel
2. Suivre **ROADMAP.md** sprint par sprint
3. Cocher dans **SPRINT-TRACKER.md** au fur et a mesure
4. Dire "Sprint X go" pour lancer un sprint

## Regles

- Chaque sprint se termine par un playtest valide
- On ne passe au sprint suivant que quand le precedent est coche
- Placeholders uniquement — pas d'assets definitifs
- Test, test, test
