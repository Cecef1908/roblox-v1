## 🎮 Phase 2 — Démo V1
**Période** : ~31 mars — ~28 avril 2026 *(~4 semaines)*

La vraie première milestone. On veut un truc jouable, même imparfait.

### Objectifs

#### Zones
- [ ] **Zone Rivière** — terrain, eau, spots d orpaillage fonctionnels
- [ ] **Zone Ville** — hub central, accès aux PNJ, shop

#### PNJ
- [ ] **Old Pete** — mentor, quêtes tutoriel
- [ ] **Rosa** — marchande, achat/vente or et équipements
- [ ] **Slim** — personnage mystérieux, quêtes secondaires

#### Core Loop
- [ ] Mécanique d orpaillage (pan/tamis) fonctionnelle
- [ ] Système de collecte de ressources (or, gemmes, déchets)
- [ ] Système de vente (joueur vers Rosa vers Robux virtuels)
- [ ] Système d achat basique (équipements de base)

#### UI/UX
- [ ] HUD minimal (or en poche, énergie, minimap)
- [ ] Inventaire basique (voir ses ressources)
- [ ] Interface shop (catalogue simplifié)
- [ ] Dialogue système PNJ (texte + son optionnel)

#### Assets
- [ ] ~15-20 modèles 3D intégrés et texturés
- [ ] Sons d ambiance basiques (eau, creusement, ville)
- [ ] Moodboard Phase 2 terminé (illustrations UI, PNJ, items)

- [ ] Screenshots pour les futurs thumbnails

## ⚠️ Risques identifiés

| Risque | Niveau | Mitigation |
|--------|--------|-----------|
| Manque de temps (side project) | 🔴 Élevé | Phases courtes, scope minimal viable, pas de scope creep |
| Compétences scripting Luau insuffisantes | 🟡 Moyen | Thorn (bot) en support, tutoriels, start avec des mécaniques simples |
| Jeu trop ambitieux, jamais fini | 🔴 Élevé | Scope V1 strict, on ne touche à rien hors scope avant la démo |
| Concurrence forte sur Roblox (mining games) | 🟡 Moyen | Différenciation par narration/PNJ + esthétique Wild West unique |
| Monétisation insuffisante | 🟡 Moyen | Étudier les game passes des concurrents avant de fixer les prix |
| Intérêt du squad qui baisse | 🟡 Moyen | Milestones courtes, petites victoires visibles, sessions de jeu ensemble |
| Assets 3D qualité insuffisante (Meshy.ai) | 🟡 Moyen | Tester Meshy tôt, avoir un plan B (marketplace Roblox, assets libres) |
