Falaises : Deux masses rocheuses enserrent la ville. Le joueur arrive par un passage etroit (~35 studs de large) entre les parois — moment de revelation quand la clairiere s'ouvre et que le Saloon apparait en face. Les falaises sont les murs naturels de la ville, aucune cloture necessaire.

Proportions : Le Saloon (22x20s = 440 studs2) domine — c'est le premier batiment visible en entrant. Le Magasin est le plus petit (14x14s = 196 studs2). Forge et Sheriff quasi identiques (~288 studs2). Espacement : ~15-20 studs entre batiments. Dimensions issues du build-library MCP (medieval/western).

Acces : Le sentier vient du pont (sud), remonte entre les falaises sur ~80 studs, puis debouche dans la clairiere. Aller simple ~15 secondes de marche.
